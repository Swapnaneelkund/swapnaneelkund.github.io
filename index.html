About Me

I’m a Full-Stack Web Developer (MERN) with hands-on experience in building scalable and real-time web applications. During my internship, I developed the complete backend for HOSLA, an NGO platform for senior citizens’ mental health, implementing secure JWT auth, REST APIs, and modular Express architecture. I also contributed to SHRADHAA, creating digital tools to support emotional well-being in elderly communities. Beyond internships, I’ve built projects like: A real-time location tracker and routing system (Socket.IO + Leaflet) A WebRTC multiplayer chess game LuxeCarry – an e-commerce app with Google OAuth, search, cart & JWT sessions Tech Stack: React.js, Node.js, Express.js, MongoDB, Socket.IO, Tailwind CSS, WebRTC, Git, Postman I’m passionate about solving real-world problems through clean, maintainable code.
